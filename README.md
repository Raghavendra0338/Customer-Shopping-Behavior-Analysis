## Customer Shopping Behavior Analysis

Welcome to the Customer Shopping Behavior Analysis project! This project focuses on analyzing customer shopping data using Python, SQL, and Power BI to discover useful insights and visualize customer purchasing patterns.

## 1. Overview

This project demonstrates the complete data analytics workflow, including:

* Loading and analyzing data in Python
* Performing Exploratory Data Analysis (EDA)
* Cleaning and preparing the dataset
* Running SQL queries for analysis
* Creating an interactive Power BI dashboard
* Generating business insights and reports

## 2. Dataset

The project uses a customer shopping behavior dataset containing information about customer purchases, shopping patterns, and product preferences.

## 3. Tools Used

* **Python** – Data analysis and cleaning
* **SQL (MySQL)** – Data querying and analysis
* **Power BI** – Dashboard creation and visualization
* **CSV Dataset** – Source data for the project

## 4. Project Steps

1. Load the dataset using Python
2. Perform Exploratory Data Analysis (EDA)
3. Clean and preprocess the data
4. Run SQL queries for deeper analysis
5. Build an interactive Power BI dashboard
6. Generate insights and visual reports

## 5. Dashboard

The Power BI dashboard includes:

* Customer purchase trends
* Product category analysis
* Sales insights
* Customer behavior visualization
* Key business metrics

## 6. Results

The analysis helps identify:

* Customer shopping patterns
* Popular product categories
* Spending behavior
* Business insights for decision-making

## 7. Files Included

* `Customer Behavior Dashboard.pbix`
* `customer_shopping_behavior.csv`
* `customer_shopping_behavior.sql`
* `Customer_Shopping_Behavior_Analysis.ipynb`

## 8. How to Run

1. Clone the repository
2. Install required Python libraries
3. Load the dataset in Python
4. Run the SQL queries
5. Open the Power BI dashboard file
6. Explore the visualizations and insights

## Conclusion

This project showcases how data analytics tools can be used to analyze customer shopping behavior and create meaningful business insights through data visualization and reporting.
